These are all Fabric mods. Download the .zip file and install fabric BEFORE you attempt to do anything with the mods.
Copy all of the .jar files into the %appdata%\.minecraft\mods folder
Copy all of the .zip files into the %appdata%\.minecraft\resourcepacks folder

If you have any issues installing these mods, google 'How to install mods using fabric'
